# TTGClub Spells to LSS CLI Parser

CLI-инструмент на Python для парсинга заклинаний D&D 5.5e с сайта [ttg.club](https://new.ttg.club) и их конвертации в форматы **LSS JSON** (совместимый с конструкторами персонажей/VTT) и чистый **Markdown**.

## 🚀 Возможности

- **Парсинг поштучно и списками**: возможность передать как имя конкретного заклинания, так и текстовый файл со списком URL-адресов.
- **Глубокий анализ механик**: автоматическое определение типа действия (`actionType`), кубов урона/лечения, спасбросков (`save`) и формул масштабирования при использовании ячеек более высокого уровня (`scaling`).
- **Очистка данных**: автоматическое удаление кастомных тегов сайта и приведение описания к чистому HTML/Markdown-виду.
- **Гибкие форматы сохранения**:
  - `lss` — индивидуальные JSON-файлы для каждого заклинания.
  - `md` — чистый Markdown с красивым оформлением метаданных.
  - `all` — сохранение обоих форматов одновременно.
  - `json-array` — объединение всех обработанных заклинаний в один общий массив JSON.

## 🛠 Установка и настройка

### 1. Клонирование репозитория и переход в папку
```bash
git clone https://github.com/Lizardeon/TTGClubSpells_toLSS_CLI.git
cd TTGClubSpells_toLSS_CLI

```

### 2. Создание и активация виртуального окружения (`venv`)

* **Windows**:
```bash
python -m venv venv
.\venv\Scripts\activate

```

* **macOS / Linux**:
```bash
python3 -m venv venv
source venv/bin/activate

```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 💻 Использование

Запуск скрипта осуществляется через терминал с передачей необходимых аргументов.

### Примеры команд:

1. **Парсинг одного заклинания (по умолчанию в формат LSS JSON):**

```bash
python TTGClub_parse_toLSS_CLI.py -s cure-wounds-phb

```

2. **Парсинг списка заклинаний из файла с объединением в один JSON-массив:**

```bash
python TTGClub_parse_toLSS_CLI.py -f list1.txt -fmt json-array -o ./output

```

3. **Сохранение заклинания во всех доступных форматах (Markdown + JSON):**

```bash
python TTGClub_parse_toLSS_CLI.py -s shield-of-faith-phb -fmt all
```

### Доступные аргументы:

* `-s`, `--spell` — Имя заклинания (часть URL после `/spells/`) или его полный URL-адрес.
* `-f`, `--file` — Путь к файлу `.txt`, содержащему список ссылок на заклинания (по одной на строку).
* `-o`, `--output` — Директория для сохранения итоговых файлов (по умолчанию `.`).
* `-fmt`, `--format` — Формат вывода. Варианты: `lss` (по умолчанию), `md`, `all`, `json-array`.
